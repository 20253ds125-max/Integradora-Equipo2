# GEDS - Gestion de eventos sociales

Proyecto web estatico para una plataforma de gestion y descubrimiento de recintos para eventos sociales.

## Vistas actuales

- `index.html`: pagina principal.
- `catalogo.html`: catalogo con busqueda, filtros por evento, capacidad y costo.
- `login.html`: inicio de sesion.
- `registro.html`: registro de usuario planner o proveedor.
- `publicar-recinto.html`: formulario para publicar un recinto nuevo.
- `detalle-recinto.html`: detalle de recinto con carrusel, amenidades y panel de disponibilidad.

## Carpetas

- `assets/css/`: estilos de cada vista.
- `assets/js/`: datos de ejemplo e interacciones.

## Como abrirlo

Abre `index.html` directamente en tu navegador. Desde el menu puedes entrar al catalogo, login, registro y publicacion de recinto.
